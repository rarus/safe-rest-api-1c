Регистр сведений «Журнал входящих HTTP-запросов»

Регистр сведений «Журнал входящих HTTP-запросов» предназначен для записи и хранения информации о параметрах входящих HTTP-запросов. 

Параметры извлекаются из входящего HTTP-запроса и представлены в удобном для чтения и анализа виде.

В журнале хранятся такие параметры запросов как: дата запроса, URL запрашиваемого ресурса, сервис, длительность обработки запроса, параметры из тела запроса и из URL, код состояния и ответ HTTP-сервиса клиенту.

Срок хранения записей журнала для каждого сервиса задается в настройках справочника "Сервисы". 

Устаревшие записи журнала удаляются по расписанию регламентного задания "Сокращение журналов входящих HTTP-запросов и конфликтов HTTP-запросов".

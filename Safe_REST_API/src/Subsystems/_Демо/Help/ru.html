Подсистема «Демо»

Подсистема «Демо» включает в себя объекты, необходимые для демонстрации основных возможностей механизма "Контроль HTTP-сервисов". На примере метода POST /partner HTTP-сервиса "_ДемоТестовыйСервис" наглядно изображено внедрение механизма контроля HTTP-запросов. Указанный HTTP-сервис можно использовать для базового тестирования возможностей механизма еще до внедрения в целевую конфигурацию.

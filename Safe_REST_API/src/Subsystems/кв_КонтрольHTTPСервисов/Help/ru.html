Подсистема «Контроль HTTP-сервисов»

Подсистема «Контроль HTTP-сервисов» включает в себя объекты, необходимые для журналирования (логирования) входящих HTTP-запросов. Также  объекты данной подсистемы позволяют обеспечить идемпотентность входящих HTTP-запросов, обнаруживать состояние гонки (race condition) и блокировать конфликтные HTTP-запросы.

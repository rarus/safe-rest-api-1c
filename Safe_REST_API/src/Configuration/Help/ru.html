Конфигурация «Контроль HTTP-сервисов. Редакция 1.0».

Инструмент предназначен для автоматизации процессов управления HTTP-сервисами на всех этапах: ведение списка HTTP-сервисов и их настройка в пользовательском режиме 1С:Предприятия, поступление запроса, контроль заполнения обязательных параметров, запись и хранение параметров запроса и ответа на него, проверка идемпотентности. Данное решение помогает организовать журналирование, логирование параметров входящих запросов и их метрик, а также организовать контроль небезопасных методов HTTP-запросов: обнаружение гонки запросов, блокирование конфликтующих запросов и их автоматическое разблокирование в соответствии с заданными параметрами.
